# Simulador-de-Imposto-de-renda
Um Arquivo no Excel feito para simular e reunir informações necessárias para a declaração de imposto de renda 
A seguinte atividade proposta feita pelo professor tinha como objetivo reunir informações de para a declaração de imposto de renda.
Todos os botões ( Títulares, Informes , Patrimônios e Montante Líquido) são clicaveis e levam direto para a aba em questão, assim como os icones do linkedin e email irão direito para o perfil da criadora desse projeto. Alem das atividades proprostas pelo professor, foi adicionado um PROCX para retornar o valor do imposto de renda da aba montante líquido.
